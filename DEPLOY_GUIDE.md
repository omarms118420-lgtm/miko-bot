# دليل نشر بوت ميكو على Koyeb مجاناً

## الخطوات

### 1. إنشاء حساب GitHub (مجاني)
- اذهب إلى: https://github.com/signup
- أنشئ حساباً مجانياً

### 2. رفع الكود على GitHub
- اذهب إلى: https://github.com/new
- أنشئ Repository باسم: miko-bot
- اضبطه على **Private** (خاص)
- في Replit، اضغط على Git ثم ارفع الكود

### 3. إنشاء حساب Koyeb (مجاني)
- اذهب إلى: https://app.koyeb.com/signup
- سجّل بحساب GitHub مباشرة
- لا تحتاج بطاقة بنكية!

### 4. نشر البوت على Koyeb
- اضغط: **Create App**
- اختر: **GitHub**
- اختر Repository: miko-bot
- اضبط:
  - Build: Dockerfile
  - Port: 5000
  - Instance Type: **nano** (مجاني)
- اضغط: **Deploy**

### 5. البوت يعمل 24/7 مجاناً!
- Koyeb يُشغّل بوتك باستمرار بدون توقف
- مجاني ولا ينام!
